# Images for The Smokey Cauldron site

Place your images here so the Home page uses them.

- **banner/** — Hero slider (e.g. `1.jpg`, `2.jpg`, `3.jpg`). Aspect ratio 21:9 or 3:1 works best.
- **best_selling/** — Best-selling items (e.g. `1.jpg` … `9.jpg`). Aspect ratio 4:3 is used.
- **gallery/** — Gallery grid (e.g. `1.jpg` … `12.jpg`). Aspect ratio 4:3 is used.

If a file is missing, the app falls back to placeholder images. Use `.jpg`, `.png`, or `.webp`.
